 Hardware wallet for Arweave.

Supports key storage, signing and verifying RSA signatures.

**WIP Use at your own risk.**

## Building

```bash
$ rustup target add riscv32imac-unknown-none-elf

```

## Supported microcontrollers

- GD32VF103
